Another Bus App

July 2015 - August 2015

Created an Android application for bus travelers using the Champaign-Urbana Mass Transit Department’s (CUMTD) public API. Utilized shared preferences to maintain a list of ‘favorite’ stops, the Google Maps for Android API to draw stop locations near the user, and developed FloatingActionButton and CardView implementations for use in the UI (prior to release of the Material Design Support Library).

The startup screen is shown on launch during a quick check for new or updated bus stops.
Up to four of the user's most recent stop selections are cached and shown when search is selected.
Favorite stops can be viewed in this fragment.

When calculation is complete, the loading indicators are replaced with the distance (in miles) to each stop.
After startup a Google Map view is shown.

Users can center on their location and select nearby bus stops.
Once a stop is selected, the 'stopwatch' view shows upcoming departures.

If the stop has multiple pickup locations, the sliding tab layout allows users to easily switch between them.

The stop can be favorited using the star in the top-right corner.
Users can also find stops by searching using CUMTD's auto-complete API.
This slide-out drawer is the app's primary navigation element.
